# Setup

1- Import FontAwesome

```html
<link
  rel="stylesheet"
  href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.2/css/all.min.css"
/>
```

2- Import CSS

```html
<link rel="stylesheet" href="components/header/header.css" />
```

3- Import JS

```html
<script defer src="./index.js"></script>
```
